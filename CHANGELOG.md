# 更新日志

## v1.0.1 (2026-06-16)

### 新增功能

- **启动认证状态检测** — 程序启动时自动检测校园网是否已认证，已认证时直接显示"已连接"状态并可注销
- **弹窗提示** — 登录/注销成功或失败时弹出居中提示窗口，需点击"确定"关闭，重要信息不再错过
- **重复登录拦截** — 已处于认证状态时点击登录会弹出提示阻止提交，避免错误操作

### 改进

- **注销按钮智能可用** — 检测到已认证状态时注销按钮自动变为可点击，无需手动打开浏览器注销
- **断线重连优化** — keep-alive 定时器先探测认证状态，已在线则跳过，减少不必要的登录请求
- **网络参数持久化** — 登录成功后将网络参数保存到配置文件，支持跨会话注销
- **密码加密存储** — 基于本机 MAC 地址派生密钥加密密码，配置文件中不再明文显示

## v1.0.0 (2026-06-15)

CQUPT 校园网登录工具首次发布。

### 新增功能

- **可视化登录界面** — 基于 tkinter 的桌面窗口，无需接触命令行即可使用
- **一键登录/注销** — 连接校园网后点击登录完成认证，点击注销断开认证
- **设备伪装** — 支持 PC / Mobile 两种设备模式，用 Mobile 模式可实现双电脑共享账号上网
- **运营商选择** — 支持中国电信、中国移动、中国联通三种运营商
- **凭据记忆** — 登录时自动保存账号密码，下次打开自动填充
- **密码显示/隐藏** — 点击眼睛图标切换密码可见性
- **开机自启** — 勾选后在 Windows 启动文件夹创建静默启动脚本，开机自动运行
- **断线自动重连** — 默认每 5 分钟自动重连，间隔可自定义，保持持续在线
- **退出选项** — 关闭窗口时可选择先注销再退出，或直接退出保持在线
- **独立可执行文件** — PyInstaller 打包为单个 exe，无需安装 Python 环境
